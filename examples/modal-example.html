Programmatic Control

You can also control modals programmatically using JavaScript.

Programmatic Control

This modal was opened programmatically using JavaScript.

// Open a specific modal Modal.open('modal-id'); // Close a specific modal Modal.close('modal-id'); // Close all open modals Modal.closeAll();

Event Handling

The modal system dispatches custom events that you can listen for.

Event log will appear here...

Event Handling

This modal demonstrates event handling. Open and close it to see events logged above.

// Listen for modal open event document.getElementById('modal-id').addEventListener('modal:open', function(e) { console.log('Modal opened!', e.detail); }); // Listen for modal close event document.getElementById('modal-id').addEventListener('modal:close', function(e) { console.log('Modal closed!', e.detail); });
